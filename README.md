<h1>docx to pdf converter</h1>
<hr>
This simple app convert word file (.docx) to .pdf file.
<hr>

<h3>Modules used →</h3>
▶ docx2pdf<br>
▶ Tkinter
<hr/>
<h3>How to run →</h3>
● Install module named 'docx2topdf'<br><br>
<pre>pip install docx2pdf</pre><br>
● run the file convert.py<hr>
